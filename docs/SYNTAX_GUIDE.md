# Syntax Guide

이 문서는 `LANGUAGE_SPEC.md`보다 읽기 쉬운 예제 중심 안내서다. 좋은 예와 나쁜 예를 함께 적어 문장형 규칙을 빠르게 익히게 하는 것이 목적이다.

## 1. 값 만들기

좋음:

```txt
이름은 "철수"이다
나이는 20이다
학생은 참이다
```

나쁨:

```txt
이름 "철수"이다
나이 = 20
학생은 참
```

설명:

- 바인딩에는 `은/는`과 `이다`가 필요하다.
- v0.1은 `=` 선언 문법을 지원하지 않는다.

## 2. 값 바꾸기

좋음:

```txt
점수는 10이다
점수를 20으로 바꾼다
```

나쁨:

```txt
점수는 10이다
점수는 20이다
```

설명:

- 같은 스코프에서 두 번째 문장은 재선언 오류다.
- 값 갱신은 `...를 ...로 바꾼다`를 쓴다.

## 3. 출력하기

좋음:

```txt
이름을 출력한다
"안녕"을 출력한다
```

나쁨:

```txt
출력(이름)
이름 출력
```

설명:

- v0.1의 기본 출력 문장은 서술형이다.
- 함수형 출력 API를 별도로 제공할 수는 있지만 언어 기본 문장은 `출력한다`다.
- 완료된 문장 뒤에는 선택적으로 마침표 `.`를 붙일 수 있다.

## 4. 조건문

좋음:

```txt
나이 > 19이면
  "성인"을 출력한다.
아니면
  "미성년자"를 출력한다.
```

나쁨:

```txt
나이 > 19 이면
  "성인"을 출력한다

if 나이 > 19
  "성인"을 출력한다

나이 > 19이면.
  "성인"을 출력한다
```

설명:

- `이면`은 조건문의 일부다.
- 영어 키워드 `if`는 지원하지 않는다.

## 5. 반복문

좋음:

```txt
인덱스는 0이다
인덱스 < 길이(숫자들)인 동안
  숫자들[인덱스]을 출력한다
  인덱스를 인덱스 + 1로 바꾼다
```

나쁨:

```txt
while 인덱스 < 길이(숫자들)
  ...
```

설명:

- v0.1 반복문은 `...인 동안` 형식 하나를 기본으로 한다.

## 6. 함수 정의

좋음:

```txt
더하기 함수는 왼쪽, 오른쪽을 받아
  왼쪽 + 오른쪽을 돌려준다
```

```txt
시작 함수는 아무것도 받지 않아
  "시작"을 출력한다
```

나쁨:

```txt
함수 더하기는 왼쪽, 오른쪽을 받아
  ...

더하기 함수(왼쪽, 오른쪽)
  ...
```

설명:

- v0.1 함수 정의는 문장형 헤더를 사용한다.
- 호출은 `더하기(1, 2)`처럼 괄호를 사용한다.

## 7. 속성 접근

좋음:

```txt
사람은 { 이름: "영희", 나이: 18 }이다
사람의 이름을 출력한다
```

나쁨:

```txt
사람.이름을 출력한다
사람의 "이름"을 출력한다
```

설명:

- 속성 접근은 `의`를 사용한다.
- v0.1에서 `의` 뒤에는 식별자만 올 수 있다.

## 8. 목록과 레코드

좋음:

```txt
숫자들은 [1, 2, 3]이다
설정은 { 호스트: "localhost", 포트: 8080 }이다
```

나쁨:

```txt
숫자들은 (1, 2, 3)이다
설정은 [호스트 = "localhost"]이다
```

## 9. 논리 연산

좋음:

```txt
성인학생은 (나이 >= 20) 그리고 학생이다
```

또 다른 예:

```txt
로그인가능은 관리자 또는 편집자이다
```

나쁨:

```txt
성인학생은 (나이 >= 20) && 학생이다
```

설명:

- 논리 연산은 `그리고`, `또는`, `아니다`를 쓴다.

## 10. 자주 틀리는 점

### 10.1 조건식은 불리언이어야 한다

좋음:

```txt
개수 > 0이면
  "있다"를 출력한다
```

나쁨:

```txt
개수이면
  "있다"를 출력한다
```

### 10.2 들여쓰기는 맞아야 한다

좋음:

```txt
참이면
  "a"를 출력한다
  "b"를 출력한다
```

나쁨:

```txt
참이면
 "a"를 출력한다
   "b"를 출력한다
```

### 10.3 재대입과 바인딩을 혼동하지 않는다

좋음:

```txt
이름은 "철수"이다
이름을 "영희"로 바꾼다
```

나쁨:

```txt
이름은 "철수"이다
이름은 "영희"이다
```
