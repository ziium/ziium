# Examples

이 문서는 v0.1 대표 예제를 모은다. 구현이 시작되면 이 예제들은 가능한 한 자동 실행 테스트와 연결하는 것이 좋다.

## 1. 기본 바인딩과 출력

```txt
이름은 "철수"이다.
나이는 20이다.

이름을 출력한다.
나이를 출력한다.
```

## 2. 재대입

```txt
점수는 10이다.
점수를 점수 + 5로 바꾼다.
점수를 출력한다.
```

## 3. 조건문

```txt
나이는 20이다.

나이 >= 20이면
  "성인"을 출력한다.
아니면
  "미성년자"를 출력한다.
```

## 4. 반복문

```txt
숫자들은 [3, 5, 8]이다.
인덱스는 0이다.

인덱스 < 길이(숫자들)인 동안
  숫자들[인덱스]을 출력한다.
  인덱스를 인덱스 + 1로 바꾼다.
```

## 5. 함수

```txt
더하기 함수는 왼쪽, 오른쪽을 받아
  왼쪽 + 오른쪽을 돌려준다.

합은 더하기(7, 8)이다.
합을 출력한다.
```

## 6. 매개변수 없는 함수

```txt
시작 함수는 아무것도 받지 않아
  "시작합니다"를 출력한다.

시작().
```

## 7. 레코드와 `의`

```txt
사람은 { 이름: "영희", 나이: 18 }이다.

사람의 이름을 출력한다.
사람의 나이를 출력한다.
```

## 8. 목록과 내장 함수

```txt
숫자들은 [1, 2]이다.
숫자들에 3 추가.
숫자들의 길이를 출력한다.
```

## 9. 단어 연산과 키워드 메시지

```txt
합은 7 더하기 8이다.
합을 출력한다.

과일들은 ["사과", "배"]이다.
과일들에 "감" 추가.
과일들의 길이를 출력한다.
```

## 10. 중첩 속성 접근

```txt
사용자는 { 이름: "민수", 주소: { 도시: "서울" } }이다.

사용자의 주소의 도시를 출력한다.
```

## 11. 잘못된 예제

아래는 오류가 나야 한다.

```txt
개수는 1이다
개수는 2이다
```

이유:

- 같은 스코프 재선언

```txt
개수이면
  "있다"를 출력한다
```

이유:

- 조건식이 불리언이 아님
