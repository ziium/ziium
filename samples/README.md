# Samples

지음 샘플 프로그램 모음이다. 아래 파일들은 현재 인터프리터로 바로 실행할 수 있다.

실행:

```bash
cargo run -- samples/00_hello_everyone.zm
```

샘플 목록:

- `00_hello_everyone.zm`: 가장 짧은 첫 출력
- `01_hello_name.zm`: 바인딩과 문자열 이어 붙이기
- `02_if_else.zm`: 조건문
- `03_countdown.zm`: 반복문과 재대입
- `04_add_function.zm`: 함수 정의와 호출
- `05_list_and_length.zm`: 목록과 내장 함수
- `06_record_profile.zm`: 레코드와 `의` 속성 접근
- `07_nested_properties.zm`: 중첩 레코드 접근
